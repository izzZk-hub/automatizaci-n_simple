# Automatización Simple: Chatbot & Agente Virtual

Este proyecto consiste en un chatbot inteligente desarrollado en Python capaz de responder consultas relacionadas con el clima actual de distintas ciudades y el precio de acciones en tiempo real. El sistema funciona desde la terminal y utiliza herramientas como Selenium, yFinance y procesamiento básico de lenguaje natural para automatizar consultas de forma sencilla y modular. La idea principal del proyecto es demostrar cómo la automatización y la inteligencia artificial basada en reglas pueden utilizarse para resolver tareas comunes mediante programación.

El chatbot fue diseñado utilizando una arquitectura modular, lo que permite separar cada funcionalidad en archivos independientes para facilitar el mantenimiento y la escalabilidad del proyecto. Dentro del directorio `funciones_agente` se encuentran las funciones principales del sistema, como `obtener_clima.py` y `obtener_precio_accion.py`, encargadas de consultar información externa. Además, el proyecto cuenta con un módulo de utilidades llamado `utils`, donde se implementa la función `sanitizar`, utilizada para limpiar y normalizar el texto ingresado por el usuario.

El funcionamiento general del chatbot es bastante simple. El usuario escribe una consulta en la terminal, por ejemplo “precio microsoft” o “clima cdmx”. Después, el sistema analiza el texto ingresado para detectar la intención de la consulta utilizando palabras clave relacionadas con clima, temperatura, acciones o precios. Una vez detectada la intención, el chatbot ejecuta automáticamente la función correspondiente y devuelve una respuesta formateada al usuario.

Para la consulta del clima, el proyecto utiliza Selenium para automatizar búsquedas en Google y extraer la temperatura actual de una ciudad específica. Selenium permite simular el comportamiento de un navegador real, por lo que el sistema puede acceder a páginas web y recuperar información dinámicamente. Por otro lado, para la consulta de acciones financieras se utiliza la librería `yfinance`, la cual permite obtener información bursátil en tiempo real directamente desde Yahoo Finance. Gracias a esto, el chatbot puede mostrar tanto el precio actual de una acción como su ticker y la divisa correspondiente, por ejemplo USD o MXN.

Otro aspecto importante del proyecto es la sanitización del texto. La función `sanitizar` convierte el texto a minúsculas y elimina caracteres especiales, tildes y variaciones del lenguaje para evitar errores al procesar las consultas del usuario. Esto permite que el chatbot pueda entender diferentes formas de escribir una misma solicitud.

El proyecto también incluye archivos de prueba como `prueba_selenium.py` y `test_chatbot.py`, utilizados para verificar el correcto funcionamiento del navegador automatizado y de las funciones principales del chatbot. Finalmente, el sistema puede ejecutarse fácilmente instalando las dependencias incluidas en `requirements.txt` y ejecutando el archivo principal `agente_selenium.py`.

Como posibles mejoras futuras, el proyecto podría integrar modelos de inteligencia artificial más avanzados, reconocimiento de voz, interfaces web o almacenamiento de historial de conversaciones, convirtiéndose en un asistente virtual mucho más completo y escalable.
